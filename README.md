> *“Parsing your data files become easier.”*

## What is this?

`bash_parsers` is a collection of bash based parsers to use in your script to parse data into bash data types, like parse csv file and store it in bash arrays system,.

---

## Dependencies

Only **Bash**, this is the idea, ~~no awk~~, ~~no grep~~, ~~no sed~~, only:

* `bash`
